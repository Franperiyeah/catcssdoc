# Welcome to CatCSS

CatCSS is a lightweight, modular, and customizable CSS framework designed to make your web development experience smooth and efficient. Whether you're building a small personal project or a large-scale application, CatCSS provides the tools you need to create stunning, responsive designs with minimal effort.

- **Lightweight**: Minimal footprint to keep your websites fast.
- **Modular**: Include only what you need, nothing more.
- **Responsive**: Built with mobile-first principles.
- **Accessible**: Following best practices for web accessibility.
- **Customizable**: Easy to override styles with your own.

---

## Installation

You can install CatCSS using **npm**:

```bash
npm install catcss
```
## Include in Your Project

Add the following line to your HTML file:

```html
<link rel="stylesheet" href="./node_modules/catcss/dist/catcss.css">
```
